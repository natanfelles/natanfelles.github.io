# Como usar seu arquivo do Twitter

A maneira mais simples de usar seu arquivo do Twitter é através da interface do navegador fornecido neste arquivo. Basta clicar duas vezes em `index.html` na pasta raiz e você poderá navegar em todo seu histórico de Tweets, dentro do seu navegador.

---

Na pasta `data`, seu arquivo do Twitter está presente em dois formatos: JSON e CSV, exportados por mês e ano.

* CSV é um formato genérico que pode ser importado para muitas ferramentas de dados, aplicativos de planilhas ou simplesmente usando uma linguagem de programação.

## JSON para Desenvolvedores

* A exportação JSON contém uma representação completa dos seus Tweets conforme retornado pela v1.1 da API do Twitter. Visite https://dev.twitter.com/docs/api/1.1 para mais informações.
* A exportação JSON também é usada para alimentar a interface de arquivo do navegador (index.html).
* Para usar a exportação em um interpretador JSON genérico em qualquer idioma, retire a primeira e última linha de cada arquivo.
